# Delta Prime Global

Luxury B2B export website for Vietnamese Artichoke Tea, Agarwood and Bird Nest, focused on Dubai, UAE and Middle East markets.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Change domain

Open `lib/site.ts` and replace:

```ts
url: "https://deltaprimeglobal.com"
```

with your real domain.

## Included

- Next.js App Router
- TypeScript
- SEO metadata
- robots.ts
- sitemap.ts
- Organization, Product, FAQ JSON-LD
- Product detail pages
- Export to Dubai landing page
- OEM / Private Label page
- RFQ form UI
- Luxury editorial design inspired by Vogue-style typography
